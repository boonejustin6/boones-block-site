# boones-block-site
Boone's Block - Rustic Country Website

A rustic country themed website with a fun vibe that's attention-catching and easy to navigate.

## Features
- Warm rustic color palette with wood tones and earth colors
- Fun elements including emojis and playful handwritten fonts
- Clear, intuitive navigation with sticky header
- Fully responsive design for mobile, tablet, and desktop
- Animated hero section with call-to-action
- Sections: Home, About, Products, Contact

## How to View
Simply open `index.html` in your web browser, or use a local server:

```bash
python3 -m http.server 8080
```

Then navigate to `http://localhost:8080`

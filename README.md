# boones-block-site

Single-page responsive site for Boone’s Block. Built to match the vintage/cream + rust palette.

Files:
- index.html
- styles.css
- script.js
- menu.md
- assets/logo.png (use provided logo image)
- assets/hero.jpg (use provided food photo)

Quick local preview:
1. Place the files in a folder and put the images under `assets/` as `logo.png` and `hero.jpg`.
2. Open `index.html` in a browser.

Deploy to GitHub Pages:
1. Create a new repository named `boones-block-site` in your GitHub account (boonejustin6).
2. From your local folder:
   - git init
   - git add .
   - git commit -m "Initial site commit"
   - git branch -M main
   - git remote add origin https://github.com/boonejustin6/boones-block-site.git
   - git push -u origin main
3. On GitHub, go to Settings → Pages and set the source to the `main` branch → `/ (root)`. Wait a minute and your site will be available at:
   https://boonejustin6.github.io/boones-block-site

# Images Directory

## How to Add Your Images

Place your image files in this directory with the following names:

### Required Images:

1. **logo.png** - Your business logo (recommended size: 200x200px or similar)
   - This will appear in the navigation header

2. **philly-hero.jpg** - Hero image of your Philly cheesesteaks (recommended size: 1200x800px)
   - This will be featured prominently on the homepage

### Optional Images:

3. **menu.pdf** or **menu.jpg** - Your full menu
   - Can be linked from the Products section

### Supported Formats:
- PNG (recommended for logos)
- JPG/JPEG (recommended for photos)
- PDF (for menu documents)

### How to Upload:

**Option 1: Via GitHub Web Interface**
1. Go to the repository on GitHub
2. Navigate to the `images` folder
3. Click "Add file" â†’ "Upload files"
4. Drag and drop your images
5. Commit the changes

**Option 2: Via Git Command Line** (if you have the repo cloned locally)
```bash
git add images/logo.png images/philly-hero.jpg
git commit -m "Add logo and product images"
git push
```

**Option 3: Share via Comment**
If you have trouble uploading, you can:
- Share a link to the images (Google Drive, Dropbox, etc.)
- Attach images in a GitHub issue or PR comment
- I can help add them for you
